Project Idea

Overview

My project idea is a mobile application to teach rhythmic notation. There are already a few applications of this type on both the Google Play and Apple App Store. However these applications only allow the user to interact by tapping the screen and dont allow the user to use what they are usually going to play rhythms with.

Motivation

Music is a method of communication like a language and like the majority of languages it is important to able to read its written form. Being able to read and perform rhythmic notation is essential to being able to send and recieve musical messages.

Description

This application will allow to user to interact with it in two ways. The first will be tapping the screen like the previously mentioned applications. The second is an audio input via the microphone on the users device. The microphone will be able to pick up the sound of the users chosen instrument.

The app will feature 'playlists' of notated rhythms either created at random with limits set by the user or by the user themselves. Once the user selects one of these 'playlists' the rhythms will appear on screen and a metronome will count in the user to begin the exercise. The BPM (beats per minute) of the metronome can also be random in a range determined by the user or at random in a 'playable' range - the BPM won't be set to 1 for example.

To more accurately detect the sound of the users instrument the user will select it from an in app menu. This selection will determine what frequencies the application will accept as an input as all instruments have limits to the frequencies of sounds they can produce. This can be seen in a number of instrument tuning applications.

Lastly the user will recieve feedback in the form of an accuracy score. This score is determined by measuring how close the users input is to the rhythm displayed and how on beat the input was. If the score is a negative number this means the user is lagging behind the beat, a positive number would indicate that the user was rushing.

Tools and Technologies

This project will require the use of mobile devices, or ways to emulate those devices, in order to test the application. The app can be developed on an Android device using a program like Android Studio. Development for the Apple App Store will the require the use of an apple device. I will need to create a developers account on each platform.

Skills Required

To program these apps I would need knowledge of Java (for Android) and Swift(for IOS). I will also have to learn XAML to create the apps UI. I may also need to learn the ins and outs of each OS to, for example, properly store user data (e.g accuracy stats) created by using the application.

Outcome

Hopefully the user will be able to use this app to faciltate their learning and allow them to be able to communicate with their fellow musicians.
